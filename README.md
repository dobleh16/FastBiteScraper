## Español

### Descripción
FastBiteScraper es una extensión de Google Chrome diseñada para la extracción de datos de restaurantes desde Google Maps. Permite recopilar información estratégica para ventas B2B, identificar prospectos potenciales y generar mensajes de contacto directo por WhatsApp.

### Características Principales
- Extracción de datos de restaurantes directamente desde Google Maps.
- Identificación automática de prospectos con WhatsApp activo.
- Cálculo de puntaje de prospecto (Score Prospecto) basado en métricas de popularidad.
- Generador de mensajes personalizados para contacto por WhatsApp.
- Exportación de datos a formato CSV compatible con Excel.

### Instalación
1. Descargue o clone este repositorio.
2. Abra Google Chrome y navegue a `chrome://extensions/`.
3. Active el "Modo de desarrollador" en la esquina superior derecha.
4. Haga clic en "Cargar extensión sin empaquetar" y seleccione la carpeta del proyecto.

### Uso
1. Realice una búsqueda de restaurantes en Google Maps.
2. Haga clic en el icono de la extensión.
3. Ingrese el nombre de la campaña si lo desea.
4. Presione el botón de extracción para recopilar los datos.
5. Utilice el generador de mensajes de WhatsApp o descargue el archivo CSV con los resultados.

## English

### Description
FastBiteScraper is a Google Chrome extension designed for extracting restaurant data from Google Maps. It allows for the collection of strategic information for B2B sales, identifying potential prospects, and generating direct contact messages for WhatsApp.

### Key Features
- Extraction of restaurant data directly from Google Maps.
- Automatic identification of prospects with active WhatsApp.
- Prospect scoring based on popularity metrics.
- Personalized WhatsApp message generator.
- Data export to Excel-compatible CSV format.

### Installation
1. Download or clone this repository.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" in the top right corner.
4. Click "Load unpacked" and select the project folder.

### Usage
1. Search for restaurants on Google Maps.
2. Click on the extension icon.
3. Enter the campaign name if desired.
4. Press the extraction button to collect data.
5. Use the WhatsApp message generator or download the CSV file with the results.
